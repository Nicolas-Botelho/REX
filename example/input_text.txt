The system is designed to help manage a small library in a simple and organized way. From the user’s perspective, it allows people to browse available books and borrow them when they are in stock. Each person has their own access to the system, where their activity is recorded, making it easy to keep track of which books they currently have and which ones they have already returned.
When someone wants to borrow a book, they can check if it is available and request it. Once the request is confirmed, the system records the borrowing and marks the book as unavailable so that others know it is already in use. Later, when the user finishes reading, they can return the book, and the system updates the record and makes the book available again for other users.
Overall, the system ensures that the library’s collection is properly managed while giving users a clear and straightforward way to borrow and return books without confusion.
